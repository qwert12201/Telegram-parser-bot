Welcome to my first telegram bot project

## ~ Features ~
1. Has `language`, `cities`, `countries` and `ruble exchange` **parsers**
2. Supports **commands**
3. Supports **update exchange**

## ~ Installation ~
Python > 3.5, probably windows os, pip package installer

1. Install repository
2. Extract files to some folder
3. Move to project directory by `cd path_to_your_folder/`
4. Use pip: `pip install -r requirements.txt`
5. Get telegram bot token from @BotFather
6. Go to **settings.py** and replace "API-TOKEN" via your token
7. Run **main.py**
Now you can use bot for your interests, if you will find the bug, please report it in Github `"Issues"`
